DPNP provides a comprehensive set of mathematical functions that mirror the NumPy API while running computations on GPU and other SYCL-enabled accelerators via Intel's oneMKL. All functions accept `dpnp.ndarray` (or `usm_ndarray`) inputs and return `dpnp.ndarray` outputs allocated in SYCL USM memory.

## Trigonometric functions

### `dpnp.sin(x, out=None, order='K')`

Computes the trigonometric sine for each element of the input array.

**Parameters:**
- `x` (dpnp.ndarray) — Input array, expected to have a floating-point data type.
- `out` (dpnp.ndarray, optional) — Output array to populate. Default: `None`.
- `order` (str, optional) — Memory layout of the output array. Default: `"K"`.

**Returns:** `dpnp.ndarray` — Element-wise sine values in radians.

```python
import dpnp as np
x = np.array([0.0, np.pi / 2, np.pi])
np.sin(x)
# array([ 0.0000000e+00,  1.0000000e+00, -8.7422777e-08])
```

## Rounding

### `dpnp.round(x, decimals=0, out=None)` / `dpnp.around(x, decimals=0, out=None)`

Rounds each element to the given number of decimal places. `around` is an alias.

**Parameters:**
- `x` (dpnp.ndarray) — Input numeric array.
- `decimals` (int, optional) — Number of decimal places. Negative values round to the left of the decimal point. Default: `0`.
- `out` (dpnp.ndarray, optional) — Output array. Default: `None`.

**Returns:** `dpnp.ndarray` — Rounded array.

```python
import dpnp as np
np.round(np.array([0.5, 1.5, 2.5, 3.5]))
# array([0., 2., 2., 4.])  # banker's rounding
```

---

### `dpnp.rint(x, out=None, order='K')`

Rounds each element to the nearest integer, rounding half values to even (banker's rounding).

**Parameters:**
- `x` (dpnp.ndarray) — Input numeric array.

**Returns:** `dpnp.ndarray` — Array rounded to nearest integer.

```python
import dpnp as np
np.rint(np.array([-1.7, -0.5, 0.2, 1.5, 2.5]))
# array([-2., 0., 0., 2., 2.])
```

---

**Returns:** `dpnp.ndarray` — Array truncated towards zero.


## Sums, products, and differences

### `dpnp.sum(a, axis=None, dtype=None, out=None, keepdims=False, initial=None, where=True)`

Sums array elements over a given axis.

**Parameters:**
- `a` (dpnp.ndarray) — Input array.
- `axis` (None, int, or tuple of ints, optional) — Axis or axes along which to sum. Default: `None` (sum all).
- `dtype` (dtype, optional) — Output data type. Default: `None`.
- `out` (dpnp.ndarray, optional) — Output array. Default: `None`.
- `keepdims` (bool, optional) — If `True`, reduced axes remain as size-1 dimensions. Default: `False`.

**Returns:** `dpnp.ndarray` — Array sum.

```python
import dpnp as np
a = np.array([[1, 2], [3, 4]])
np.sum(a)          # array(10)
np.sum(a, axis=0)  # array([4, 6])
np.sum(a, axis=1)  # array([3, 7])
```

---

### `dpnp.prod(a, axis=None, dtype=None, out=None, keepdims=False, initial=None, where=True)`

Returns the product of array elements over a given axis.

**Parameters:**
- `a` (dpnp.ndarray) — Input array.
- `axis` (None, int, or tuple of ints, optional) — Axis or axes. Default: `None`.
- `dtype` (dtype, optional) — Output type. Default: `None`.
- `keepdims` (bool, optional) — Keep reduced dimensions. Default: `False`.

**Returns:** `dpnp.ndarray` — Product result.

```python
import dpnp as np
np.prod(np.array([1, 2, 3, 4]))
# array(24)
```

---

### `dpnp.cumsum(a, axis=None, dtype=None, out=None)`

Returns the cumulative sum of elements along a given axis.

**Parameters:**
- `a` (dpnp.ndarray) — Input array.
- `axis` (None or int, optional) — Axis along which to accumulate. Default: `None` (flattened).
- `dtype` (dtype, optional) — Output type. Default: `None`.

**Returns:** `dpnp.ndarray` — Cumulative sum with the same shape as `a` (or 1-D if `axis=None`).

```python
import dpnp as np
a = np.array([[1, 2, 3], [4, 5, 6]])
np.cumsum(a)           # array([ 1,  3,  6, 10, 15, 21])
np.cumsum(a, axis=0)   # array([[1, 2, 3], [5, 7, 9]])
```

---

### `dpnp.diff(a, n=1, axis=-1, prepend=None, append=None)`

Calculates the n-th discrete difference along the given axis.

**Parameters:**
- `a` (dpnp.ndarray) — Input array.
- `n` (int, optional) — Number of times to difference. Default: `1`.
- `axis` (int, optional) — Axis to difference along. Default: `-1`.
- `prepend`, `append` (scalar or dpnp.ndarray, optional) — Values to prepend/append before differencing. Default: `None`.

**Returns:** `dpnp.ndarray` — Differenced array; the axis dimension is reduced by `n`.

```python
import dpnp as np
np.diff(np.array([1, 2, 4, 7, 0]))
# array([ 1,  2,  3, -7])
```

---

### `dpnp.gradient(f, *varargs, axis=None, edge_order=1)`

Returns the gradient of an N-dimensional array using central differences in the interior and first/second order differences at boundaries.

**Parameters:**
- `f` (dpnp.ndarray) — Input array.
- `*varargs` (scalar or dpnp.ndarray) — Spacing between samples. Default: unit spacing.
- `axis` (None, int, or tuple of ints, optional) — Axes to compute gradient along. Default: all axes.
- `edge_order` (int, optional) — 1 or 2; order of accuracy at the boundaries. Default: `1`.

**Returns:** `dpnp.ndarray` or list of `dpnp.ndarray` — Gradient arrays.

```python
import dpnp as np
f = np.array([1, 4, 9, 16], dtype=float)
np.gradient(f)
# array([3., 4., 6., 7.])
```

---

### `dpnp.cross(a, b, axisa=-1, axisb=-1, axisc=-1, axis=None)`

Returns the cross product of two (arrays of) vectors in R³.

**Parameters:**
- `a` (dpnp.ndarray) — First input array; last axis defines vectors.
- `b` (dpnp.ndarray) — Second input array.
- `axisa`, `axisb`, `axisc` (int, optional) — Axis defining vectors in `a`, `b`, and output. Default: `-1`.
- `axis` (int, optional) — Overrides `axisa`, `axisb`, and `axisc`. Default: `None`.

**Returns:** `dpnp.ndarray` — Vector cross products.

```python
import dpnp as np
np.cross(np.array([1, 2, 3]), np.array([4, 5, 6]))
# array([-3,  6, -3])
```

---

### `dpnp.trapezoid(y, x=None, dx=1.0, axis=-1)`

Integrates along the given axis using the composite trapezoidal rule.

**Parameters:**
- `y` (dpnp.ndarray) — Input array to integrate.
- `x` (dpnp.ndarray, optional) — Sample points corresponding to `y`. Default: `None` (uniform spacing).
- `dx` (scalar, optional) — Spacing when `x` is `None`. Default: `1.0`.
- `axis` (int, optional) — Axis to integrate along. Default: `-1`.

**Returns:** `dpnp.ndarray` — Definite integral approximation.

```python
import dpnp as np

```

### `dpnp.nan_to_num(x, copy=True, nan=0.0, posinf=None, neginf=None)`

Replaces NaN with zero and Inf with large finite numbers.

**Parameters:**
- `x` (dpnp.ndarray) — Input array.
- `copy` (bool, optional) — Create a copy. Default: `True`.
- `nan` (float, optional) — Replacement for NaN. Default: `0.0`.
- `posinf` (float, optional) — Replacement for positive infinity. Default: largest float.
- `neginf` (float, optional) — Replacement for negative infinity. Default: most negative float.

**Returns:** `dpnp.ndarray` — Array with NaN and Inf replaced.

```python
import dpnp as np
x = np.array([np.nan, np.inf, -np.inf, 1.0])
np.nan_to_num(x)
# array([ 0.00000000e+000,  1.79769313e+308, -1.79769313e+308,  1.00000000e+000])
```

## Miscellaneous

### `dpnp.clip(a, min=None, max=None, *, out=None, order='K')`

Clips (limits) values in an array to the range `[min, max]`.

**Parameters:**
- `a` (dpnp.ndarray) — Array containing elements to clip.
- `min` (None, scalar, or dpnp.ndarray, optional) — Minimum value. Default: `None`.
- `max` (None, scalar, or dpnp.ndarray, optional) — Maximum value. Default: `None`.
- `out` (dpnp.ndarray, optional) — Output array. Default: `None`.

**Returns:** `dpnp.ndarray` — Clipped array.

```python
import dpnp as np
a = np.arange(10)
np.clip(a, 1, 8)
# array([1, 1, 2, 3, 4, 5, 6, 7, 8, 8])
```

### `dpnp.interp(x, xp, fp, left=None, right=None, period=None)`

Returns the one-dimensional piecewise linear interpolation to a function with given discrete data points `(xp, fp)`.

**Parameters:**
- `x` (dpnp.ndarray) — x-coordinates at which to evaluate the interpolated values.
- `xp` (dpnp.ndarray) — x-coordinates of the data points (must be increasing).
- `fp` (dpnp.ndarray) — y-coordinates of the data points.
- `left` (float, optional) — Value below `xp[0]`. Default: `fp[0]`.
- `right` (float, optional) — Value above `xp[-1]`. Default: `fp[-1]`.
- `period` (float, optional) — Period for periodic interpolation. Default: `None`.

**Returns:** `dpnp.ndarray` — Interpolated values, same shape as `x`.

```python
import dpnp as np
xp = np.array([1.0, 2.0, 3.0])
fp = np.array([3.0, 2.0, 0.0])
np.interp(np.array([1.5, 2.5]), xp, fp)
# array([2.5, 1.0])
```

### `dpnp.modf(x, out=None)`

Returns the fractional and integral parts of an array, element-wise.

**Parameters:**
- `x` (dpnp.ndarray) — Input real-valued array.
- `out` (tuple of dpnp.ndarray, optional) — Output arrays for fractional and integral parts.

**Returns:** `(dpnp.ndarray, dpnp.ndarray)` — Fractional part and integral part.

```python
import dpnp as np
np.modf(np.array([-1.7, 0.3, 2.9]))
# (array([-0.7,  0.3,  0.9]), array([-1.,  0.,  2.]))
```

### `dpnp.frexp(x, out=None)`

Decomposes elements into mantissa and exponent of a base-2 representation: `x = mantissa * 2^exponent`.

**Parameters:**
- `x` (dpnp.ndarray) — Input floating-point array.

**Returns:** `(dpnp.ndarray, dpnp.ndarray)` — Mantissa in `[-1, -0.5) ∪ [0.5, 1)` and integer exponent.

```python
import dpnp as np
np.frexp(np.array(1.0))
# (array(0.5), array(1))
```

---

### `dpnp.ldexp(x1, x2, out=None, order='K')`

Returns `x1 * 2**x2` element-wise. This is the inverse of `frexp`.

**Parameters:**
- `x1` (dpnp.ndarray or scalar) — Mantissa array.
- `x2` (dpnp.ndarray or scalar) — Integer exponent array.

**Returns:** `dpnp.ndarray` — `x1 * 2^x2`.

```python
import dpnp as np
np.ldexp(np.array([0.5, 0.5]), np.array([2, 4]))
# array([2., 8.])
```
