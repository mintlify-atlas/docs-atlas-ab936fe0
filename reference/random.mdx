title: "Random number generation — dpnp.random"
description: "GPU-accelerated random number generation for DPNP. Covers RandomState, seeding, simple random data, permutations, and 35 probability distributions."
---

The `dpnp.random` module provides GPU-aware random number generation that mirrors the `numpy.random` legacy API. Generators run on a SYCL device using the MT19937 (Mersenne Twister) algorithm. Most functions accept a `device` and `sycl_queue` keyword so that outputs can be created directly on a specific accelerator without a host-to-device copy. When a particular distribution is not yet accelerated, the module falls back transparently to NumPy.

## Seeding and the RandomState class

### `dpnp.random.seed(seed=None, device=None, sycl_queue=None)`

Reseed the MT19937 random number generator engine for the given device.

**Parameters:**
- `seed` (`None`, int, or array-like of at most 3 ints, optional) — Seed value. `None` uses a random seed. Default: `None`.
- `device` (None, str, SyclDevice, SyclQueue, or Device, optional) — Target device. Default: `None`.
- `sycl_queue` (SyclQueue, optional) — SYCL queue for the operation. Default: `None`.

```python
import dpnp as np

np.random.seed(42)
x = np.random.rand(5)
# reproducible result each run
```

---

### `class dpnp.random.RandomState(seed=None, device=None, sycl_queue=None)`

Container for the MT19937 pseudo-random number generator, scoped to a specific SYCL device.

**Parameters:**
- `seed` (`None`, int, or array-like of at most 3 ints, optional) — Initial seed. Default: `None`.
- `device` (None, str, SyclDevice, SyclQueue, or Device, optional) — Device where the generator lives.
- `sycl_queue` (SyclQueue, optional) — SYCL queue to use.

Instantiating `RandomState` directly lets you maintain independent generators on different devices or control seeding precisely.

```python
import dpnp as np

rng = np.random.RandomState(seed=0)
samples = rng.normal(loc=0.0, scale=1.0, size=(3, 3))
```

## Simple random data

### `dpnp.random.rand(*args, device=None, usm_type="device", sycl_queue=None)`

Create an array of the given shape filled with random samples from a uniform distribution over `[0, 1)`.

**Parameters:**
- `*args` (sequence of ints) — Shape dimensions.
- `device` / `sycl_queue` — Target device/queue. Default: `None`.

**Returns:** `dpnp.ndarray` — Float64 (or float32 on devices without float64) random array.

```python
import dpnp as np

np.random.rand(3, 2)
# array of shape (3, 2) with values in [0, 1)
```

---

### `dpnp.random.randn(d0, *dn, device=None, usm_type="device", sycl_queue=None)`

Return samples from the standard normal distribution `N(0, 1)`.

**Parameters:**
- `d0, *dn` (ints) — Shape dimensions.

**Returns:** `dpnp.ndarray` — Float64/float32 array of standard normal samples.

```python
import dpnp as np

s = 3 + 2.5 * np.random.randn(2, 4)
# samples from N(3, 6.25)
```

### `dpnp.random.randint(low, high=None, size=None, dtype=int, device=None, usm_type="device", sycl_queue=None)`

Return random integers from `low` (inclusive) to `high` (exclusive).

**Parameters:**
- `low` (int) — Lowest integer to draw (inclusive); upper bound when `high` is omitted.
- `high` (int, optional) — Upper bound (exclusive). Default: `None`.
- `size` (int or tuple, optional) — Output shape. Default: `None` (single value).
- `dtype` (`dpnp.int32` or `int`, default `int`) — Output dtype.

**Returns:** `dpnp.ndarray` — Integer array of shape `size`.

```python
import dpnp as np

np.random.randint(3, 11, size=1000, dtype=np.int32)
```

### `dpnp.random.choice(a, size=None, replace=True, p=None)`

Generate a random sample from a 1-D array.

**Parameters:**
- `a` (int or 1-D array-like) — If int, samples are drawn from `arange(a)`.
- `size` (int or tuple, optional) — Output shape.
- `replace` (bool, default `True`) — Sample with replacement.
- `p` (array-like, optional) — Probabilities for each element.

**Returns:** `dpnp.ndarray` — Random samples from `a`.

<Note>
  This function currently uses `numpy.random.choice` on the backend.
</Note>

## Permutations

### `dpnp.random.shuffle(x1)`

Modify a sequence in-place by shuffling its contents along the first axis.

**Parameters:**
- `x1` (dpnp.ndarray) — Array to shuffle in-place.

**Returns:** `None`

```python
import dpnp as np

arr = np.arange(10)
np.random.shuffle(arr)
```

---

### `dpnp.random.permutation(x)`

Randomly permute a sequence or return a permuted range. Unlike `shuffle`, this does not modify the input and always returns a new array.

**Parameters:**
- `x` (int or dpnp.ndarray) — If int, permute `arange(x)`; otherwise permute `x`.

**Returns:** `dpnp.ndarray` — Permuted array.

```python
import dpnp as np

np.random.permutation(10)
# array([3, 8, 7, 9, 0, 6, 1, 2, 4, 5])  # random
```
