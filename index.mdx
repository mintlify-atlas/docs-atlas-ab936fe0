DPNP (Data Parallel Extension for NumPy) is a Python library that implements a NumPy-compatible API which runs on any SYCL-capable data-parallel device — including Intel integrated and discrete GPUs, NVIDIA GPUs, and AMD GPUs. You can often replace `import numpy as np` with `import dpnp as np` and immediately benefit from GPU acceleration without rewriting your code.

## Why DPNP?

DPNP brings high-performance, data-parallel computing to Python without requiring you to learn new APIs. It leverages Intel's oneAPI toolkit and oneMKL to accelerate numerical workloads on modern hardware.

## Get started

<Steps>
  <Step title="Install DPNP">
    Install using conda or pip from the Intel channel:
    ```bash
    conda install dpnp -c https://software.repos.intel.com/python/conda/ -c conda-forge --override-channels
    ```
  </Step>
  <Step title="Import and use">
    Replace your NumPy import and run computations on the default device:
    ```python
    import dpnp as np

    x = np.array([1, 2, 3, 4, 5])
    print(np.sum(x))   # Executes on GPU if available
    ```
  </Step>
  <Step title="Select a device">
    Explicitly target a specific device by passing `device=` to any array constructor:
    ```python
    import dpnp as np
    import dpctl

    gpu_device = dpctl.select_gpu_device()
    x = np.linspace(0, 1, 1000, device=gpu_device)
    result = np.sin(x)
    print("Result on:", result.device)
    ```
  </Step>
  <Step title="Explore the API">
    Browse the [API Reference](/reference/array-creation) for the full list of supported functions, or read the [migration guide](/guides/migrating-from-numpy) to understand compatibility.
  </Step>
</Steps>
